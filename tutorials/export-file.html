Overview

The ExportFile plugin allows exporting the table content to a file, blob or string.
The plugin is always enabled, so there is no additional configuration involved. See the How it works section to learn more about how it works under the hood.

How it works

The plugin allows you to export data from Handsontable in one of available format.
Currently we have only one predefined strategy. You can export file as in the CSV format.

See available methods to learn which will help you in exporting data from Handsontable.

Available methods

The plugin exposes three public methods to export data.

  • downloadFile(format, options) - allows you to generate downloadable file, directly in user browser.
  • exportAsBlob(format, options) - allows you to export JavaScript Blob object.
  • exportAsString(format, options) - allows you to export data as a string.

All of them accept the same arguments:

format String

Format is required to prepare predefine settings object. Currently we implemented only 'csv' format.

options Object

It's a optional argument. Contains set of supported option and extends predefined CSV configuration. See available options to learn what options you can use.

Available options in the export configuration

Below you can find all supported options:

bom Boolean

Allows to export data with BOM signature.

Please note: This property will prepend content by UTF-16BE BOM signature (FE FF). Browser will convert signature to UTF-8 value (EF BB BF) automatically.

You can use this property in all of the available methods.

Default value: true

columnDelimiter String

Allows to define columns delimiter.

You can use this property in all of the available methods.

Default value: ','

columnHeaders Boolean

Allows to export data with their column header.

You can use this property in all of the available methods.

Default value: false

exportHiddenColumns Boolean

Allows to export data from hidden columns.

You can use this property in all of the available methods.

Default value: false

exportHiddenRows Boolean

Allows to export data from hidden rows.

You can use this property in all of the available methods.

Default value: false

fileExtension String

Allows to define file extension.

You can use this property in downloadFile method.

Default value: 'csv'

filename String

Allows to define file name.

You can use predefined placeholders, which will be replaced by date.

You can use this property in downloadFile method.

Default value: 'Handsontable [YYYY]-[MM]-[DD]'

mimeType String

Allows to define MIME type.

You can use this property in downloadFile and exportAsBlob methods.

Default value: 'text/csv'

range Array

Allows to define range of dataset to export. It's represented by an array of numeric, visual indexes [startRow, startColumn, endRow, endColumn].

You can use this property in all of the available methods.

Default value: 'text/csv'

rowDelimiter String

Allows to define rows delimiter.

You can use this property in all of the available methods.

Default value: '\r\n'

rowHeaders Boolean

Allows to export data with their row header.

You can use this property in all of the available methods.

Default value: false
