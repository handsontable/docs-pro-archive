Overview

The Filters plugin allows filtering the data in the table's columns using a range of pre-defined conditions.

This plugin uses the Trim Rows plugin to hide the filtered out rows. It allows not rendering these rows at all, which is good for the overall table performance.

Quick setup

To enable the plugin you need to set the filters property to true and enable the filters dependency, the dropdownMenu plugin.

Custom dropdown menu

To display only filters while hiding the other elements in the dropdown menu, pass the elements to be displayed as an array in the configuration.
